# Student Performance Management System

A web-based student performance management system designed to manage students, teachers, subjects, marks, and study activities efficiently.

## Features

### Admin Module
- Admin Login
- Manage Students
- Manage Teachers
- Manage Subjects
- Assign Subjects to Teachers

### Teacher Module
- Teacher Login & Account Activation
- Semester and Section Selection
- Load Students by Section
- Enter Student Marks
- Automatic Percentage Calculation
- Pass/Fail Evaluation
- Mentoring Marks Entry
- Input Validation for Incorrect Marks

### Student Module
- Student Registration & Login
- View Marks and Mentoring Details
- Daily Study Activity Tracking
- Subject-wise Study Recording
- Bar Graph and Pie Chart Performance Analysis

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
